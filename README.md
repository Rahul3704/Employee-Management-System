# Employee Management System (EMS)

A backend-based **Employee Management System** built using **Node.js and Express**.  
This project manages employees, authentication, stats, and data export using a JSON-based database.

## Features
- Employee CRUD operations
- Authentication (Login system)
- Employee statistics
- Export employee data
- Middleware-based route protection

## Tech Stack
- Node.js
- Express.js
- JavaScript
- JSON (as database)
- HTML, CSS (basic frontend)
